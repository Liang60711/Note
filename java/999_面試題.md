## 描述 Bean 生命週期

1. 實例化 Bean : 
    - 使用反射或Factory方法創建。

2. 填充 Bean 屬性、依賴注入 : 
    - populateBean() 注入依賴。

    - 循環依賴由 Spring 的 三級緩存（singletonObjects、earlySingletonObjects、singletonFactories）解決。

3. 調用 aware 接口相關的方法 : 
    - 包括 BeanNameAware.setBeanName()、BeanFactoryAware.setBeanFactory()、ApplicationContextAware.setApplicationContext()。

    - 發生在屬性注入完成之後，初始化前。

4. 調用 BeanPostProcessor 前置處理方法 : 
    - postProcessBeforeInitialization()。

    - 可以對 Bean 進行修改或包裝。

5. 初始化 (Initialization) : 

    - 執行順序：

        1. @PostConstruct 方法

        2. InitializingBean.afterPropertiesSet() 方法

        3. @Bean(initMethod="...")（如果有設定）


6. 調用 BeanPostProcessor 後置處理。

    - postProcessAfterInitialization()

    - 常用於 AOP 或代理包裝。

7. Bean 就緒，可被使用

8. Spring 容器關閉時，Bean 被銷毀。

    - @PreDestroy 方法

    - DisposableBean.destroy()

    - @Bean(destroyMethod="...")（如果有設定）
