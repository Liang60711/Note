## 正規表達式
字元集
```python 
[abc]           # 包含abc
[abc{]          # 包含abc，符號

[A-Z]           # 包含英文大寫
[a-z]           # 包含英文小寫
[a-zA-Z]        # 包含英文
[0-9]           # 包含數字0-9

[^abc]          # 不包含abc，[^] 表示之外
```


```python
\w              # 包含英文字、數字、底線，即 [A-Za-z0-9_]
\W              # \w以外，即 [^A-Za-z0-9_]

\d              # 即 [0-9]
\D              # 即 [^0-9]

\s              # 空白字元，即 [\n\t\r\f]
\S              # 即 [^\n\t\r\f]
```
