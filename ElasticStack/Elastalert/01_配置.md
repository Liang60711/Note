## 配置

目錄結構

```
elastalert
|─ elastalert.yaml
└─ rules
    └─ rule-1.yaml
    └─ rule-2.yaml
```




配置 `/opt/elastalert/elastalert.yaml`

```yml
# 規則存放的目錄
rules_folder: /opt/elastalert/rules
# 多久執行一次檢查
run_every:
  seconds: 10
# 會檢查前五分鐘內是否有因為網路延遲而漏掉的數據
buffer_time:
  minutes: 5

# Elasticsearch 連線資訊
es_host: 192.168.110.128
es_port: 9200
es_username: 'elastic'
es_password: '@WSX3edc'

# elastalert2 套件產生的日誌，寫進 elasticsearch 指定的 index
writeback_index: elastalert_status

# 超過2天的日誌就不觸發警報
alert_time_limit:
  days: 2
```
