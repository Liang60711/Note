## 安裝 / 開啟
1. windows 下載安裝檔
2. 安裝後，開啟 redis-server

    ```sh
    redis-server
    ```

    如果打不開，需要先去 redis-cli 關閉舊連線

    ```sh
    # 打開 cli
    redis-cli

    # 關閉
    shutdown
    ```
