# 條件渲染

## v-show

v-show 是填入一個 boolean，本質是使用 `display:none`，這個 CSS。

```html
<div id='root'>
    <h2 v-show="true"> Hello World </h2>
</div>
```

<br/>

<br/>

## v-if 

v-if 是填入一個 boolean，本質是，是否要顯示這個 dom 元素。

```html
<div id='root'>
    <h2 v-show="true"> Hello World </h2>
</div>
```

<br/>

<br/>

## v-show 和 v-if 比較

兩者的差別是: `是否會建立 dom 元素`。

* v-show: 會產生 dom 元素，但會隱藏。

* v-if: 不會產生 dom 元素。

`重點` : 如果是很頻繁的切換，建議是使用 `v-show`，效能會比較差 (dom元素的增減)。


<br/>

<br/>

## v-if v-else-if v-else

用法和 js 相同，但這個結構`不允許被打斷`，一定要是緊鄰的。

```html
<div id='root'>
    <h2 v-if="n === 1"> 111 </h2>
    <h2 v-else-if="n === 2"> 222 </h2>
    <h2 v-else-if="n === 3"> 333 </h2>
    <h2 v-else> 444 </h2>
</div>
```

如果被打斷，打斷的元素後面都不生效
```html
<div id='root'>
    <h2 v-if="n === 1"> 111 </h2>

    <h2> @打斷 </h2>

    <h2 v-else-if="n === 2"> 222 </h2>
    <h2 v-else-if="n === 3"> 333 </h2>
    <h2 v-else> 444 </h2>
</div>
```

<br/>

<br/>


## template + v-if 

如果需要把一個區塊，都用 v-if 來判斷，可以使用 template

```html
<div id='root'>
    <template v-if="n === 1">
        <h2> 111 </h2>
        <h2> 222 </h2>
        <h2> 333 </h2>
    </template>
</div>
```

重點: 
1. 渲染時，dom 中不會顯示 template 標籤。

2. template 只能搭配 `v-if` 使用，不可以搭配 `v-show`。
